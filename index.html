In the article section "Has design become too hard?", Zeldman discusses how now a days most designers are unsatisfied with what they do and that nothing is perfect for most designers, today it takes a lot of focus to master the art of coding. Tools have developed, new ones have been discovered and has become more complicated for web designers. One problem with web design was people owning different platforms. Many of the screens were different sizes so it became difficult to create something that was required to be universal and adjustable to all sizes of different monitors/platforms.

For an example, the web design would adjust to a PC but not for a Mac. To fix this problem liquid layout was to help work around the different types monitors and help it act as one. Web design today has been made more convenient but nothing is ever easy, there will always be room for improvement. This is my active voice fix.
